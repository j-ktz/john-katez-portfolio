# John Katez Portfolio (Static Site)

This is a fast, accessible HTML/CSS portfolio site optimized for recruiter scanning and basic SEO.

## Free hosting options

### Option A — GitHub Pages (recommended)
1. Create a GitHub account (if you don’t have one).
2. Create a new repository named: `john-katez-portfolio` (any name is fine).
3. Upload *all files* from this folder to the repo root.
4. In GitHub: Settings → Pages → Build and deployment:
   - Source: Deploy from a branch
   - Branch: main / root
5. Your site will publish at: `https://<your-username>.github.io/john-katez-portfolio/`

### Option B — Cloudflare Pages
1. Create a Cloudflare account (free).
2. Pages → Create a project → Upload assets.
3. Upload the folder contents.

## Updating with real visuals
- Add screenshots to each case study page by inserting `<img>` tags.
- Put images in `assets/img/` and reference them like: `assets/img/ask-alfredo-1.png`
